SCREAM assistant: Root Cause Analysis for Cyber-Security.

Getting started

This tool is built around two main components: The SCREAM analysis tool that is used to investigate known attacks and build attack modes, and the Socio-Technical Threat Modeller (STTM) which aims at using results from past SCREAM analyses to predict new situations.

If it's the first time that you start the application you need first to take care of your Socio-Technical Capabilities repository by clicking on the following button or by clicking on the 'SCREAM' tab in the navigation bar. Take me to SCREAM analysis

Once you performed or imported several SCREAM analyses, you can you the STTM by clicking the following button or selecting 'STTM' in the navigation bar. Take me to the STTM
